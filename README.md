# Mental Rotation experiment

This repository contains an experiment on mental rotation of objects, created with <a href="https://magpie-ea.github.io/magpie-site/">_magpie</a>. It was part of the Experimental Psychology Lab 2021 of the University of Osnabrück.

The images used here are materials from <a href="http://doi.org/10.5334/jopd.ai">Ganis & Kievit (2015)</a>.
